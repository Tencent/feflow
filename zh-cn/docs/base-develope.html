开始开发

项目创建好后，在项目目录下运行 npm i 安装好依赖，然后再运行 feflow dev 就能启动本地服务，服务默认运行在 http://127.0.0.1:8001。

打开 http://127.0.0.1:8001 ，你会看到初始态的一个页面。如果你想修改这个页面，则需要先大概了解一下项目的目录结构，知道去哪改。
