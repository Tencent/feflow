title: 命令
---

## init

``` bash
$ feflow init
```

创建一个项目。

## dev

``` bash
$ feflow dev
```

本地开发。

## build

``` bash
$ feflow build
```

生产环境打包。

## lint

``` bash
$ feflow lint [folder]
```

对某个目录下的Javascript文件进行规范检查。

## install

``` bash
$ feflow install [package]
```

安装脚手架、插件。

## config 

``` bash
$ feflow config <key> <value>
```

配置CLI客户端，包括npm 的 registry和proxy。
